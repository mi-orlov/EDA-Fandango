# Разведочный анализ данных (EDA): Исследование аномалий рейтингов Fandango

Исходные данные для статьи [Be Suspicious Of Online Movie Ratings, Especially Fandango’s](http://fivethirtyeight.com/features/fandango-movies-ratings/) находятся в свободном доступе по следующей ссылке на github: https://github.com/fivethirtyeight/data. Это два csv-файла. В одном файле данные о рейтингах от компании Fandango, а в другом файле агрегированные данные о рейтингах от других компаний, таких как Metacritic, IMDB и Rotten Tomatoes.

Проект посвящен статистическому анализу и проверке результатов статьи  о намеренном завышении оценок к фильмам со стороны сайта Fandango. В ходе работы было проведено сравнение скоринговых систем Fandango с другими крупными платформами (Metacritic, IMDB, Rotten Tomatoes) на основе исторических данных.

### Цель проекта
Математически проверить, существует ли сдвиг в распределении оценок фильмов на сайте Fandango, и обосновать выводы с помощью методов визуализации и агрегации данных.

### Стек технологий
* **Язык программирования:** Python
* **Библиотеки:** Pandas, NumPy, Seaborn, Matplotlib

### Этапы работы:
1. **Предобработка и очистка данных:** Импорт сырых датасетов, фильтрация данных, работа с числовыми признаками.
2. **Feature Engineering:** Приведение рейтингов разных платформ к единой шкале (нормализация оценок к 5-балльной системе) для корректного сопоставления.
3. **Разведочный анализ (EDA):** Расчет описательных статистик (среднее, медиана, дисперсия, экстремальные значения) для каждого сайта.
4. **Визуализация данных:** Построение графиков распределения (KDE plots), гистограмм и диаграмм рассеяния (Scatter plots) для сравнительного анализа смещения рейтингов.

### Результаты и выводы:
* **Подтверждение гипотезы:** Анализ распределения плотности оценок (KDE) и вычисления основных характеристик математически доказали явный сдвиг рейтингов Fandango в большую сторону по сравнению с конкурентами.
* **Аномалия распределения:** Средний балл на Fandango стабильно превышает 3.0. На платформе практически полностью отсутствуют фильмы с откровенно низкими оценками, что указывает на искусственное завышение или округление округляемых значений.

### Описание признаков

Анализ проводится на основе двух таблиц. Ниже представлено описание ключевых колонок:

**1. Таблица `fandango_scrape.csv` (Данные, выгруженные с Fandango):**
| Колонка | Описание |
| :--- | :--- |
| `FILM` | Название фильма |
| `STARS` | Количество звёзд на Fandango.com |
| `RATING` |  Рейтинг Fandango - значение, прочитанное с HTML-страницы. Это средний рейтинг фильма. |
| `VOTES` | Количество голосов пользователей, которые написали отзыв о фильме (на момент выгрузки данных). |

**2. Таблица `all_sites_scores.csv` (Сравнение с другими платформами):**
| Колонка | Описание |
| :--- | :--- |
| `FILM` | Название фильма |
| `RottenTomatoes` | Оценка критиков (Tomatometer) |
| `RottenTomatoes_User` | Оценка обычных пользователей Rotten Tomatoes |
| `Metacritic` | Оценка критиков Metacritic |
| `Metacritic_User` | Оценка пользователей Metacritic |
| `IMDB` | Оценка пользователей платформы IMDb |
| `Metacritic_user_vote_count` | Количество проголосовавших пользователей на Metacritic |
| `IMDB_user_vote_count` | Количество проголосовавших пользователей на IMDb |
